This website and set of tools was created as a project within the Bioinformatics program at Seneca College.

The primary contributors to this project were Katelyn Davis, Kathy Lam, Nina Nissan, Rene Quevedo & Philip Rees.

Several different programming languages and existing tools were used in order to create this project. Here is a brief summary of the languages and packages used:

  • The main architecture for of project was done using Django web framework
  • The database management system used was MySQL
  • Website design was done following HTML5 & CSS3 standards
  • The Contig Retreival and Gene Annotation tools were written in Perl and the open reading frame graphics that are generated was created with the BioPerl Library
  • The local BLAST search funciontality uses BioPython
  • The FragGeneScan software was used to generate predictions of the ORFs
  • The tools created are can be computationally intensive and to make runtimes manageable we use the Sharcnet cluster of computers to carry out the computations.

The website is best viewed in either Chrome or Firefox.

It is best to use a screen resolution of 1024x768 or greater.

We would like to thank Trevor Charles and all lab members for helping to oversee this project and all of his reccommendations. Additionally, thank you to the Science Computing group here at the University of Waterloo for hosting the site.
