Overview

The Metagenomic Clone Bank is currently the Charles Lab's primary data management system and custom anlaysis pipeline for cosmid clones that have been isolated from metagenomic libraries. The metagenomic libraries used in screening have been constructed using DNA from environmental sources and include, but are not restricted to, those from the Canadian MetaMicrobiome Library collection.

Clone information

Metagenomic (or genomic) libraries are subjected to function-based screening to isolate individual clones. Cosmid clones can be logged in this database along with their relevant metadata and sequence information. This database also includes information for Subclones, and biochemical assays that have been carried out on clones.

Automated sequence retrieval and gene annotation

Custom tools are available to aid in the analysis of clones, including a tool for retrieval of clone sequences from assembled pooled sequencing data, as well as automated ORF identification and annotation. Please see the Tools pages for details.
